# 一、核心观点总结
- 居民收入端中位数增幅5%，消费结构未明显改善，基础性消费占比高。
- 美联储新主席候选人表现中立，倾向于维持美联储职能。
- 中国一季度居民消费和收入水平分析，城镇居民可支配收入是农村居民的两倍以上。
- 能源生产量分析，煤炭产量基本无增，天然气增速较大，火电和水电增长明显。
- 大宗商品市场整体表现不佳，黑色金属低位横盘，有色金属电解铝创新高，化工产品整体回落，新能源电池材料价格波动。
- 农产品市场生猪价格持续下跌，豆粕价格大跌，农药价格因需求增加而上涨。
- 美元利率强势，但人民币相对美元走强，反映市场对美元信用衰退的担忧。
- 市场对美联储4月底不降息的概率极高。

# 二、方法论总结
- **选股逻辑**：关注基础性消费、能源行业、大宗商品、农产品、化工产品等，寻找长期核心资产和行业周期性变化。
- **买入策略**：根据行业景气度和价格波动情况，寻找买入时机，如行业底部和长期增长潜力。
- **卖出策略**：根据价格走势和行业基本面变化，设置止盈和止损点。
- **风控方式**：分散投资，关注行业整体趋势，控制仓位。
- **流派**：趋势交易和价值投资，结合周期性交易和基本面分析。
